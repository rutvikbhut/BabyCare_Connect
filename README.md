# BabyCare Connect

**BabyCare Connect** is a specialized platform designed to streamline communication and management between parents and childcare providers. This repository contains the core logic and interface components necessary to create a cohesive environment for tracking baby activities, health data, and daily schedules.

## 🚀 Features

*   **Real-time Activity Tracking**: Log feedings, sleep patterns, and diaper changes instantly.
*   **Secure Communication**: Integrated messaging system for parents and caregivers.
*   **Health Dashboard**: Monitor growth charts, vaccination schedules, and medical appointments.
*   **Account Management**: Seamlessly switch between parent and provider roles with secure session handling.
*   **Multi-device Support**: Responsive interface designed for both desktop and mobile use.

## 📁 Project Structure

The project is organized to ensure scalability and ease of maintenance:

*   `/src`: Contains the primary source code for the application.
*   `/public`: Static assets including images, icons, and the base HTML template.
*   `/docs`: Detailed documentation regarding API endpoints and UI components.

## 🛠️ Installation & Setup

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/rutvikbhut/BabyCare_Connect.git
    ```
2.  **Navigate to the project directory**
    ```bash
    cd BabyCare_Connect
    ```
3.  **Install dependencies**
    ```bash
    npm install
    ```
4.  **Start the development server**
    ```bash
    npm start
    ```

## 📝 Usage

Once the server is running, you can access the interface via `localhost:3000`. Users can sign up as either a **Parent** or a **Caregiver** to begin managing baby care profiles. The interface supports standard Markdown for notes and logs, ensuring clear and formatted communication.
